# Hotel & Airbnb Price Comparator Chrome Extension

A Chrome Extension that compares hotel & Airbnb prices side by side while browsing.

## Features
- Auto-detect hotel listings on major booking sites
- Real-time price comparison across multiple platforms
- AI-powered travel recommendations
- Price drop alerts
- Historical price charts

## Tech Stack
- Frontend: React.js + TailwindCSS
- Backend: Node.js + Express + MongoDB
- Chrome Extension: Manifest V3
- Authentication: Firebase Auth

## Project Structure
- `/client` - Chrome Extension (React UI + Content Scripts)
- `/server` - Node.js Backend API
